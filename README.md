ECG Heart Disease Detection System

This project analyzes ECG signals and detects heart disease using machine learning.

Technologies:
Python, NumPy, Pandas, Scikit-learn, SciPy, Tkinter, pyserial

How to run:
1. Install libraries: pip install -r requirements.txt
2. Run: python src/ecg_gui_analysis_autolabel.py
